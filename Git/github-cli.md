The `gh` tool is used to do anything and everything GitHub related on the command line.

### Authentication

Use `gh auth login` command to set up authentication.
Go through the steps to login and you'll be able to use `gh` to manage GitHub through the command line.

### Getting help

There's a lot of help text for `gh`.
Run any `gh` command with --help to see usage, flags and examples.

```
gh --help
gh repo --help
gh repo create --help
```
