# Vim cheat sheet

- `J` deletes the line break between two lines.
- `:%!expand -t2` substitute tabs with 2 spaces on throughout the file.
- `help <something> | only` make the current window the only one on the screen. Useful when I try to read a help page without splitting the window.
- `ea` append to end of a world.
- `t<char>` to navigate to first character in line
